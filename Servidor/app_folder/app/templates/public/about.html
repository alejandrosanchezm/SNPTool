Sobre la aplicación

SNP Tool es una aplicación de clasificación de Polimorfismos de nucleótido único (SNP) basada en algoritmos de Inteligencia Artificial. La aplicación utiliza como base unos metapredictores de los siguientes 'scores':

  • BayesDel Score. Los datasets con los Scores precalculados pueden extraerse del siguiente link.
  • ClinPred Score. Se puede descargar un dataset con algunos Scores precalculados del siguiente link.

Estos metapredictores utilizan Random Forest Regressor como modelo para la predicción de los valores que deberíamos obtener.

Los metapredictores NO son los modelos originales; es decir: nos proporcionan un valor aproximado del valor que deberían tomar estos scores y que sirven para el modelo para obtener el resultado final.

Una vez calculados estas puntuaciones para cada uno de los inputs, se utilizan en la entrada del modelo de clasificación, que utiliza un Random Forest Classifier como modelo de inteligencia artificial.

Random forest gif, de Ryan Holbrook

Finalmente, se completa la información mostrada con una llamada a la siguiente API de VEP para mostrar más información útil sobre las búsquedas realizadas.
