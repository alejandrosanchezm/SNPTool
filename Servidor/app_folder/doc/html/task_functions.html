task_functions module¶

Task Functions: incluye funciones que son usadas por las tareas.

task_functions.posprocess(x)¶

Transforma los datos que han sido utilizados en los modelos para que puedan ser mostrados por pantalla. Devuelve los datos transformados.

Parámetros

x (Pandas DataFrame.) – Pandas DataFrame con los datos para ser procesados

Devuelve

Devuelve el DataFrame procesado.

Tipo del valor devuelto

Pandas DataFrame.

task_functions.requestToVest(x)¶

Realiza una llamada a la API de VEST del siguiente enlace: https://rest.ensembl.org/documentation/info/vep_region_get y extrae la información “transcript_consequences” y “most_severe_consequence” para cada una de las filas de los resultados. Devuelve None en caso de que la llamada no devuelva un valor correcto.

Parámetros

x (Pandas Series.) – Pandas Series (fila) del dataframe con datos para hacer la petición a VEP.

Devuelve

Devuelve un diccionario con los campos transcript_consequences y most_severe_consequence en caso de que consiga respuesta; en caso contrario, devuelve None.

Tipo del valor devuelto

Dictionary en caso de éxito, None en caso contrario.
